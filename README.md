2. 작업중 격였던 애로사항

    에뮬레어터에서 작업할때 필요 자바 버전을 수시로 변경해야 하고 다른 버전을 요구해서 에뮬려에터에서 작업을 수행하지 못함

3. 작업 방식

    에뮬레이터에서 작업을 할 수 없으므로 USB에 휴대폰을 연결하여 작업
